JavaScript forEach() Function Description. JavaScript forEach() is a JavaScript array method. forEach() calls a provided callback function once for each element of the array. It can be only used on Arrays, Map and Sets Datatypes of the JavaScript. The provided callback function can perform any type of operation on the element of the given array. The forEach() method calls a function once for each element in an array, in order. Note: the function is not executed for array elements without values.
